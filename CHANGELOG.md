# 更新日志

## v1.0.0

### 🚀 新特性
- ⚡ **流式输出** — 回答内容逐段实时显示，无需等待全部完成
- 🔧 **工具调用可视化** — 实时显示 AI 正在调用的工具（如 `read_file`、`execute_shell_command` 等）
- 📍 **自动感知工作区** — AI 自动识别 VS Code 当前打开的文件夹路径
- 💬 **右侧对话面板**，支持对话记忆
- 💡 **解释代码** — 解释选中代码的功能和逻辑
- ⚡ **优化代码** — 分析问题并提供优化方案
- 🔍 **找问题** — 审查代码，找出潜在隐患
- 📋 **发送选中代码**到对话框
- 🗑️ **清空对话**

### 🔧 技术
- 基于 QwenPaw HTTP API（SSE 流式响应）
- 支持 session_id 会话记忆
- VSIX 双击安装
